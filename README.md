# Quiz Mastery

Quiz Mastery is a comprehensive Flutter application designed to test your knowledge across multiple categories and difficulty levels.

## Features
- Multiple Quiz Categories
- Adaptive Difficulty Levels
- High Scores & Leaderboards
- Clean, Modern UI

## Getting Started
To run this project:
1. Ensure Flutter SDK is installed.
2. Run `flutter pub get` to fetch dependencies.
3. Run `flutter run` to launch the application.
